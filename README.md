# Крысы и двери

Здесь вы найдёте правила для настольной приключенческой игры "Крысы и двери" (или просто КиД), являющейся ретроклоном OD&D (нулевой или оригинальной редакции D&D 1974 года).

В репозитории лежат .md файлы и сделанные из них .pdf файлы.

Версия КиД [0.96](https://github.com/StuartZaq/rats-and-doors/tree/ver.0.96) является наиболее близкой к пересказу оригинальной версии D&D. В последующих версиях КиД добавлены так называемые хоумрулы и версии прочтения оригинальных правил.

# Pandoc

pandoc 3.9.0.2  
XeTeX 3.141592653-2.6-0.999998 (TeX Live 2026)   
pdfTeX 3.141592653-2.6-1.40.29 (TeX Live 2026)

PDF сделаны из .md файлов с помощью терминала и команды:

`./build_pdfs.sh`, которая запускает генерацию сразу нескольких pdf.

По-одному файлу можно создать с помощью команды типа:

`pandoc %файл%.md   --metadata-file=version.yaml   --metadata-file=common.yaml   --pdf-engine=xelatex -s   -o "%название файла%.pdf"`

- `--metadata-file=version.yaml` берет во внимание yaml часть, касающуюся номера версии и даты создания файлов pdf
- `--metadata-file=common.yaml` аналогично тому, что выше, но касается формата текста, шрифта и т.п.
- `--pdf-engine=xelatex` добавлено чтобы терминал/pandoc поняли кириллические символы в .md файлах

Шрифт Futura PT нужен с расширенным кол-вом символов, иначе кавычки-ёлочки могут не отрисоваться. В common.yaml вы можете поменять шрифт на другой.
