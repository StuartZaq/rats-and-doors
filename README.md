# Крысы и двери

Здесь вы найдёте правила для настольной приключенческой игры "Крысы и двери" (или просто КиД), являющейся ретроклоном OD&D (нулевой или оригинальной редакции D&D 1974 года).

В репозитории лежат .md файлы и сделанные из них .pdf файлы.

Версия КиД [0.96](https://github.com/StuartZaq/rats-and-doors/tree/ver.0.96) является наиболее близкой к пересказу оригинальной версии D&D. В последующих версиях КиД добавлены так называемые хоумрулы и версии прочтения оригинальных правил.

# Pandoc

pandoc 2.14.2
XeTeX 3.14159265-2.6-0.999992 (TeX Live 2020)
pdfTeX 3.14159265-2.6-1.40.21 (TeX Live 2020)

PDF сделаны из .md файлов с помощью терминала и команды:

`pandoc --pdf-engine=xelatex rules.md -s -o "Rats and Doors ver.0.97.pdf"`

`--pdf-engine=xelatex` добавлено чтобы терминал/pandoc поняли кириллические символы в .md файлах. 

Шрифт Futura PT нужен с расширенным кол-вом символов, иначе кавычки-ёлочки могут не отрисоваться. В YAML части .md файлов вы можете поменять шрифт на другой.
